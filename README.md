# TreinoBarb

Tracker pessoal de treino — planilha semanal, registro de séries, evolução de cargas, medidas corporais, mobilidade e alongamento.

## Rodar localmente

```
python start.py
```

Abre automaticamente em `http://localhost:8081/index.html`.

## Acessar de qualquer lugar

Via GitHub Pages (HTTPS):

```
https://[usuario].github.io/TreinoBarb/
```

A autenticação por senha funciona apenas em HTTPS ou localhost (Web Crypto API). No celular, use sempre a URL do GitHub Pages.

## Armazenamento

Os dados ficam no `localStorage` do navegador, por dispositivo. Treinos salvos no celular não aparecem no PC (e vice-versa).
